# power\_pv\_forecasts

Generating forecasts for our clients' renewable energy sources for submission to Distribution System Operators' platforms.







1. Hirschvogel:

a) uruchom skrypt pv\_forecast\_model

Jeśli zadziała prawidłowo, powinien w folderze EWE\\SP\_Dzial\_RH - Zasoby\\Projekty\\SOGL prognozowanie\\pliki\_SOGL umieścić 4 pliki:

* RRRRMMDD\_MMDD\_HVG\_33P0009.xml
* RRRRMMDD\_MMDD\_HVG\_33P0010.xml

* RRRRMMDD\_MMDD\_HVG\_33P0009.csv

* RRRRMMDD\_MMDD\_HVG\_33P0010.csv



b) zaloguj się na platformę https://logowanie.tauron-dystrybucja.pl/



c) W sekcji PLANY kliknij "Import z pliku - MWE"



d) Wybierz typ planu: "generacji" oraz dowolnie - csv lub xml. Pliki csv i xml zawierać powinny te same dane.



e) w zależności od wybranego typu pliku, załącz pojedynczo albo csv, albo xml



f) zatwierdź, klikając "DODAJ PLIK, WYBIERAJĄC GO Z DYSKU"



g) czynności c-f należy wykonać dwukrotnie, tak aby dodać plan zarówno dla 33P0009, jak i 33P0010



h) sprawdź komunikaty na platformie lub mail od td.pgbsogl@tauron-dystrybucja.pl - czy poprawnie przyjęto dane.
